Meet Drought

Drought is a HTML & CSS barebones, boilerplate, framework or whatever else you want to call it. Why introduce another HTML/CSS project like this? There are so many boilerplates/frameworks out there but this is what works for me when starting a new site. Many of the great things here have come from other boilerplates including H5BP. I wanted something a little more then a boilerplate and something that grow about from writing the same code over and over. The major addition being that this project includes CSS for a two column web site with a header and footer. A lot of the sites we do fit this style and this starting point lets me then easily change the width of each column and get started on the unique styling for the site instead of having to get the barebones structure/layout out of the way first.

This project will most likely not be right for you or your work flow. I believe that everybody should have their own boilerplate/framework to start projects with. It keeps from coding the same stuff over and over and helps you get started faster on your project and allows you to have some guidelines to follow. I don't expect this project to work for you but I hope you get inspiration to create your own or take pieces from here that do work.
